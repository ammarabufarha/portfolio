## Customization

To customize this portfolio:

1. **Colors**: Edit CSS variables in `styles.css` (`:root` selector)
2. **Content**: Update HTML content in `index.html`
3. **Animations**: Modify animation keyframes in `styles.css`
4. **Styling**: Adjust component styles in `styles.css`
